<<Level 2>> ssDNA Repair Pathways[1]

• There are 3 primary mechanisms by which a cell fixes single-strand breaks: mismatch repair pathway (MMR), nucleotide excision repair (NER), and basic excision repair (BER)

• The third of these pathways is where the PARP1, 2, and 3 enzymes are involved and are primarily responsible for fixing the single-strand breaks by the mechanism of BER

<<Level 2>> The Biology of PARP Inhibition[8]

• Poly(ADP-ribose) polymerase (PARP)1 is a key enzyme involved in fixing single-strand DNA damage via the basic excision repair pathway

o It binds directly to sites of single-strand damage, and once activated by the single-strand damage it uses nicotinamide adenine dinucleotide (NAD) to add these branching side chains of PARP; this is called PARylation

o This process results in the recruitment of a whole host of other enzymes into this protein complex; if the DNA single-strand damage is not repaired, apoptosis may ensue, and if the damage is repaired the cancer cell may survive

<<Level 2>> PARP Inhibition Can Lead to Increase dsDNA Breaks[8]

• Inactivation or inhibition of PARP can prevent recruitment of enzymes to the sites of single-stand breaks

o As these single-strand breaks accumulate and go unrepaired, there can be an arrest at the replication fork causing in turn a double-strand DNA break

<<Level 2>> Synthetic Lethality Hypothesis[9,10]

• This graphic shows how synthetic lethality works in the context of PARP inhibition

o In the first scenario there is a normal cell without a BRCA mutation, with normal PARP and BRCA function

▪ This cell is able to repair both single-strand and double-strand DNA breaks and if a PARP inhibitor is used, PARP function is inhibited, but the normal BRCA function will be able to restore the DNA repair function and that cell will remain healthy and not die

• In the second scenario, we have a cell that has a monoallelic or a single copy lost or inactivation of a BRCA gene

o Because these are tumor suppressor genes, both copies need to be inactivated for the BRCA function to be lost

o With a single copy or monoallelic mutation, the PARP function could be inhibited by a PARP inhibitor, but the BRCA function will be restored and maintained because the second allele, which is a normal allele, will recover and compensate for the faulty allele

• In the third scenario in the pretreatment situation, the BRCA function is obliterated with both alleles lost, leaving behind only the PARP function as the only mechanism by which this cell can fix DNA damage

o In this scenario if a PARP inhibitor is used, the BRCA function is already lost because of the biallelic BRCA2 mutation and now we are inhibiting this cell's only other mechanism that can be used to repair DNA damage and therefore this results in cell death

• The key message is that it is important that both alleles or both copies of the BRCA gene or other genes are inactivated for PARP inhibitors to work best

<<Level 2>> MMR Mutations Can Cause HRD Mutations!

• A primary mismatch repair mutation may lead to a secondary mutation in a homologous repair gene

• In this case study, the patient has a MSH2 mutation, stop mutation E809X, and a loss of heterozygosity of the second allele, a true biallelic loss

o Because of this primary genetic event a number of secondary genetic events have taken place including a frame shift mutation in BRCA2, ERCC4 and 5, FANCM and MSH6

o This patient should not receive treatment with a PARP inhibitor despite the BRCA mutation because this is not the driving event; the primary event in this case is the MSH2 lesion and this patient should probably receive a programmed cell death-1 (PD-1) inhibitor
